Title Screen

The title screen contains five options.

  • New Game
  • Continue
  • Options
  • Quit
  • Wishlist the next game

In the background is an image of Siffrin falling backwards accompanied by darkless particle effects.


Main Menu

The main menu contains seven options.

  • Pockets
  • Craft
  • Equip
  • Status
  • Options
  • Save
  • Quit

In the backround there is an image of the party members.

There is also a panel showing the number '145' in the bottom left. This is an artifact from RPGMaker MV representing the current amount of money held by the party.


Pockets

The Pockets menu contains several submenus.

  • Pockets
  • Knick-Knacks
  • Memories

Craft

The Craft menu allows you to select a character in the order of Traveler-Fighter-Housemaiden-Researcher and see a list of their known Craft Skills, along with their level, HP, and Charge.


Equip

First, you select a character, in the usual order.

The Equip menu is split into five windows.



On the top, it displays the description of the selected equipment


On the upper right, it displays:

  • Equip
  • Optimize
  • Clear

Neither Optimize nor Clear have functionality.



On the lower right, it displays:

  • Weapon
  • Equipment
  • Accessory

On the left, it displays:

  • Max HP->
  • Attack->
  • Defense->
  • Atk.Speed->
  • Luck->
  • CritChance->
  • HP Regen->
  • ExpRate->

On the bottom, it displays nothing.


Status

First, you select a character in the usual order.

The status menu is split into 4 parts


The top shows the selected character's title


The top middle shows the selected character's:

  • LVL
  • HP
  • Charge
  • Current EXP
  • To Next Level

The bottom middle shows the selected character's:

  • Attack
  • Defense
  • C. Attack
  • C. Defense
  • Agility
  • Luck

Along with their equipment.


The bottom shows a description of the character.

The Traveler

It's you.

The Fighter

A fighter from the city of Jouvante. He quit his job to join in this journey.

The Housemaiden

A housemaiden from the town of Dormont, blessed by the Change God.

The Researcher

A researcher from Ka Bue. Her craft knowledge is unparalleled.

Save

Selecting Save will bring you to a menu with 4 panels.

On the left, the twenty four save slots are displayed, numbered as "File 1", "File 2", and so on.

The message in the top panel reads:

Please select a file slot.


Save files

Save files display some information about themselves in the bottom right panel when hovered over.

START AGAIN START AGAIN START AGAIN: a prologue v1.7

The icons of the four combat-able party members are displayed directly below, followed by their levels.

At the bottom is the current playthrough time.



Unused save files display as "(...)" when hovered



Selecting a save file allows you to choose in the middle right panel:

  • Load
  • Save
  • Delete

Save

Saves the current progress in your game.


If a file is selected:

Do you wish to overwrite this save file?


Load

Loads the data from the saved game.


If a file is selected:

Do you wish to load this save file?

  • Yes
  • No

Delete

Deletes all data from this save file.


If a file is selected:

Do you wish to delete this save file?

  • Yes
  • No

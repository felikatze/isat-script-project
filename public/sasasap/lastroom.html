Last Room / Snack Room

Text for the last room before facing the King in START AGAIN.

The True End triggers at 10+ Weird points, the Perfect End at 4 or less.

Any Weird Point value between the two, as well as the first loop of the game, results in the regular ending(s).


Text for seeing True End before is only possible to obtain if you died battling the King on the True End path. Text for the regular endings and the Perfect End are identical, unless stated otherwise.


"Galre" is Adrienne's typo, not mine. Check your game files. It's there.

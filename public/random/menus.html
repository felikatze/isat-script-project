Title Screen

The title screen contains four options.

  • New Game
  • Continue
  • Options
  • Quit

The animation of Siffrin on the title screen changes depending on the ACT.

  • ACT 1 (Eye closed, sleeping)
  • ACT 2 (Eye open, with shine)
  • ACT 3 & 4 (Eye open, without shine)
  • ACT 5 (Face shadowed)
  • ACT 6 (Eye closed, smiling)
  • After credits (Empty field)

Main Menu

The main menu contains seven options.

  • Pockets
  • Craft
  • Equip
  • Profile
  • Settings
  • Load
  • Quit

Pockets

The Pockets menu contains several submenus.

  • Pockets
  • Equipment
  • Memories
  • Souvenirs

Craft

The Craft menu leads to a list of all currently known Craft skills sorted by character, in the usual order of Siffrin-Mirabelle-Isabeau-Odile-Bonnie.


Equip

First, you select a character, in the usual order of Siffrin-Mirabelle-Isabeau-Odile-Bonnie.

The Equip menu is split into two windows.



On the left, it displays:

  • Weapon
  • Equipment
  • Memory

On the right, it displays:

  • Max HP
  • Attack
  • Defense
  • Atk.Speed
  • Luck
  • CritChance
  • HP Regen
  • ExpRate

When changing any equipment, the stats also show how they will be changed by the new equipment, with the original stats to the left, and an arrow pointing at the changed stats to the right.


Profile

The "Profile" menu is separated into two tabs. The left, titled "Profile", shows the character's profile.


The right, titled "Stats", shows the selected character's:

  • Title
  • LVL
  • HP
  • Attack
  • Defense
  • Atk Speed
  • Luck
  • Current EXP
  • To Next Level

Options / Settings

All

A list of all of the game's settings.

General

General settings that alter the way the game behaves.


Language

Choose the game's language.

Only available on the title screen.


Available settings:

  • EN
  • JP

Fullscreen

Toogle between fullscreen or windowed mode.


Always Run

Player walks when OFF. Player runs when ON.


Available settings:

  • OFF
  • ON

CommandRemember

Game remembers the last command selected during battle.


Available settings:

  • OFF
  • ON

Message Speed

Changes the speed text is displayed during messages.


Can be increased in incriments of 1. Minimum is 1, maximum is 10.


Fast-Forward

Fast forward through all fast-forwardable conversations, regardless of whether you've seen them once before.

Only available on New Game+.


Available settings:

  • OFF
  • ON

Audio

Adjust the audio settings for the game.


All audio settings adjust in incriments of 10%. Minimum is 0%, maximum is 100%.


Master Volume

Adjusts the overall volume of the game.


BGM Volume

Adjusts the volume of the background music.


SE Volume

Adjusts the volume of the sound effects.

Visual

Settings that adjust the visual properties of the game.

All visual settings only have the options OFF and ON.


When accessed from the title screen

(Not accessible on Title)


Screen Effects

Turn OFF to disable screen shakes and flashes during dialogue and battles.


Text Shake

Turn OFF to disable shaking text during dialogue.


BG Scroll

Turn OFF to disable the scrolling background during battles and on the map.


Battle Help

Turn ON to add a Rock Paper Scissors Cheat Sheet during battles.

Controls

Changes the way you can control the game.


Gamepad Config

Configure the game's gamepad settings. (Only works if gamepad is plugged in.)


The following inputs can be reconfigured:


OK / Interact

Confirm menus/Interact with the world.


Cancel

Cancel menu actions.


Run

Hold to run on the field.


Menu

Access the Main Menu.


L

Quickly scroll up menus.


R

Quickly scroll down menus.


Other options in this menu include:


Reset to Default

Returns your controller to default settings.


Finish Configuration

Are you done configuring your gamepad?


Keyboard Config

Configure the game's keyboard settings.

Default Keyboard Layout

Reverts your keyboard setting to default setup.


WASD Movement Layout

Changes your keyboard to WASD movement.


Finish Configuration

Are you done configuring your keyboard?


Hovering over a key

Change the configuration of this key?


Button Prompts

Manually change which button prompts appear during gameplay.


Available settings:

  • PC
  • PS
  • XBOX

Exit

Exit the Options Menu.


Load / Save

Menus accessed via saving at a Save Point or using the Load menu are identical in layout.

Up top, the twelve save slots are displayed, numbered as "Save 1", "Save 2", and so on.

The message above them reads:

Please select a save slot.


Below the save slots are three options.

  • Load
  • Save
  • Delete

Load

Loads the data from the saved game.


If a file is selected:

Do you wish to load this save file?

  • Yes
  • No

Save

Saves the current progress in your game.

Greyed out if selected from the "Load" menu.


If a file is selected:

Do you wish to overwrite this save file?


Delete

Deletes all data from this save file.


If a file is selected:

Do you wish to delete this save file?

  • Yes
  • No

Save files

Save files display some information about themselves when hovered over.

IN STARS AND TIME

ACT [number]


  • LOOP [number]
  • PLAYTIME [XX:XX:XX]
  • MAP [map]
